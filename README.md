# 💰 Costs - Gestão Empresarial

O **Costs** é uma aplicação de gestão de projetos e controle de custos, desenvolvida para consolidar meus conhecimentos em **React.js**. Este projeto faz parte da minha jornada de formação acadêmica, com o objetivo claro de me tornar um **Desenvolvedor Front-End**.

> 💡 Projeto baseado no curso de React do Matheus Battisti, com personalizações minhas próprias de estilização, estrutura de pastas e lógica de componentes.

---

## 🚀 Funcionalidades

- [x] Criação e gerenciamento de projetos.
- [x] Adição de serviços de orçamento.
- [x] Edição e exclusão de dados (CRUD).
- [x] Interface interativa.

## 🛠 Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes ferramentas:

- **React.js** (Hooks, Router, Components)
- **JavaScript** (ES6+)
- **Framer Motion** (Biblioteca de animação)
- **CSS Modules** (Estilização escopada)
- **HTML5**
- **CSS3**

## 📈 Padrão de Commits

Para este projeto, adotei o padrão de **Conventional Commits** para manter o histórico de desenvolvimento organizado e profissional:

| Tipo         | Descrição                                          |
| :----------- | :------------------------------------------------- |
| **feat**     | Adição de um novo recurso ou funcionalidade        |
| **fix**      | Correção de um bug                                 |
| **docs**     | Atualização de documentação                        |
| **style**    | Mudanças de formatação/estilo, sem afetar a lógica |
| **refactor** | Refatoração de código                              |
| **test**     | Adição ou modificação de testes                    |
| **chore**    | Atualizações de build ou ferramentas auxiliares    |

## 🔧 Como rodar o projeto

Siga os passos abaixo para testar a aplicação localmente:

1. **Clone o repositório:**

   ```bash
   git clone https://github.com/guilhermefpo/costs.git
   ```

2. **Instale as dependências:**

   ```bash
   npm i
   ```

   ou

   ```bash
   npm install
   ```

3. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```
